Overview

You can perform multiple operations from the client side :

  • Create/Read/Update/Delete entities (CRUD operations)
  • Call entity/dataclass/collection methods
  • Authenticate users and get user information
All asynchronous methods return a promise, so you'll have to be familiar with it to use the client connector.

Any events declared on the server side are also called if the associated event was triggered by one of the performed operations.

All data manipulation operations are restricted to authorized users. You can learn more about the subject in the security section.
