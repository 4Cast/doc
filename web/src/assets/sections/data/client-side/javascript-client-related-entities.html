Related entities

The JavaScript Client allows you to set, retrieve and access related entities.

Preloading

When retrieving entities from the server, you can use the expand option to force the server to include some related entities or related collections in its response.

Lazy loading

If you have already retrieved an entity from the server but then needed to load a related entity or a related collection due to some user action, you can use the fetch() method.
