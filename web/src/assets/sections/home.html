Welcome to Wakanda guide

Wakanda is a full stack javascript platform for coding web, hybrid, native applications and going in production. It's composed of two products:

  • Studio: creates your application (frontend and backend)
  • Server: hosts and runs your application in production and development

Wakanda studio provides:

  • Scaffolding: your application for
    • backend in javascript or typescript
    • web angular v2 or v4
    • mobile ionic v2
  • Code Editor: integrates Ace web editor
  • Code completion: on your code, on all wakanda APIs and on your application data models
  • Debugger: integrates chrome devtools for debugging ssjs code even from a node worker

Wakanda server provides:

  • Data Integration: uses built-in database or integrates other database
  • Authentication: uses built-in authentication mechanism or overrides it with your session management module
  • HTTP: configure your HTTP server and add your own HTTP endpoints
  • Workers: runs your JS code on threads for asynchronous behaviour. Use node worker with all Node LTS modules
  • Administration: manages your running server and application in production
