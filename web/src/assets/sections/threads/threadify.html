Threadify

What is it

  • A helper module to simplify the usage of NodeWorkers and SharedWorkers.

Usage

Threadify is a module that manages a pool of workers for you.
  • You can execute module methods in a worker with ease.
  • You can select between NodeWorkers and SharedWorkers as execution context.
  • By default, all modules share the available workers.
  • You can dedicate a worker to a specific module.
