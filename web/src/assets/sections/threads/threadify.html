Threadify

What is it

  • It simplifies the use of NodeWorker and SharedWorker.

Usage

Threadify is a module that manages a pool of worker for you.
  • It can execute module methods in a worker.
  • It handles a pool of workers.
  • It handles NodeWorkers and SharedWorkers.
  • It can dedicate a worker to a module
