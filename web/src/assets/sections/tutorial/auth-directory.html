User authentication is managed by the Directory.

The Directory is the part of Wakanda Server that handles user creation, removal, login, logout as well as assigning users to specific groups which allows to build a security system that includes Authentication and Authorization.

The user is identified by a login/password combination that has already been defined in the Directory.

To access the directory, simply click on the little man on the solution explorer.

Go ahead and create two users : Jenna Does and John Doe

You can add users later on using the server side API.

So far we have talked about authentication, the other part of an application security is authorization, which is the mechanism of giving a user the privileges to access a resource.

Wakanda allows to do so by creating groups and assigning users to them You can then associate permissions to a specific group.
