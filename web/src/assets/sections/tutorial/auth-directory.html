Backend Authentication

The Directory is the part of Wakanda Server that handles user creation, removal, login, logout, assigning users to specific groups and more.

In our TODO app we will identify every user by a login/password combination that has already been defined in the Directory editor inside Wakanda Studio.

Later on, you can learn about adding users using the Directory API.

To access the directory editor, click on the little man icon on the solution explorer.

Go ahead and create two users :

  • Login Name : jane.doe Full Name : Jane Doe Password : jane
  • Login Name : john.doe Full Name : John Doe Password : john

Let's also create a todoUsers group and add our two newly created users to it. This will come in handy in the next steps.

Later on, we will see how to use this group in order to secure access to a resource.
