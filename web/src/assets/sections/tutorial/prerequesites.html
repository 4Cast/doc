Prerequisites

In order to follow along, all you’ll need is a wakanda studio that can be downloaded from here.

With the aim of providing a set of tools and best practices to enhance productivity, kickstarting a project is made easy with wakanda scaffolding system.

To scaffold an application, pick the project type you are interested in building and the files will be generated for you. Focus on your application and let us worry about the details.

There are many different things that can cause a problem with your application startup, for example if a dependency is missing or an admin privilege is required. Don’t panic we’ve got your back. Once an issue is detected it will be displayed in the studio console thanks to the troubleshooting feature.

Troubleshooting allows you to locate the cause of different issues that you might encounter as well as providing hints to solve them.

After resolving all the issues detected by the studio troubleshooting, click the Run button. Behind the curtains an npm install followed by an npm start will be triggered.

you can select the project types you want to run.

Keep your application running and transpiling while we go through the tutorial.
