Overview

Wakanda provides a way to run JavaScript code and system processes in separate threads :

  • Three types of workers are available: SystemWorker, SharedWorker, NodeWorker
  • A Worker proxy is an object that helps you communicate with the worker during its lifetime.

Use a :

  • System Worker to run an external system application (ex: html2pdf, shell, ...).
  • Node Worker to start a Node JS thread that will run code in the background and that you can communicate with from anywhere in your app.
  • Shared Worker to start a Wakanda JS thread that will run code in the background and that you can communicate with from anywhere in your app.
