How to add new Model

To work with a new model do a right click on the Model folder.


You can choose to add :

  • A new local Model - This can be useful to split your model into different parts
  • A virual Model - Use this kind of model to map a remote web service/ API (ckeck the dedicated section for more information
  • A MySQL Model to work with a Remote MySQL
  • An ODBC Model to interface with ODBC external datasources
  • Wakanda or 4D to work with a remote 4D or Wakanda Server

Just fill the connection properties and click ok

Filling the dialog, actually create a config.waConfig file which is a JSON format file. Advanced user can work directly (create/edit) this file without using the dialog
