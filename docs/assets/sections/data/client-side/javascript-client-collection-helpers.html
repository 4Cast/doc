Handle Collections

All collections returned by the JavaScript client are Arrays that expose three methods :
  • nextPage(): get the next page of the collection
  • prevPage(): get the previous page of the collection
  • more(): load a new page and append it to the current collection

The difference between more and nextPage is that the first will append the retrieved entities to the array, where nextPage will replace the array's content with the retrieved entities.

All three methods return a promise and change the collection's content.
