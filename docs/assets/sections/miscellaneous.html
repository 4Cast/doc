What ES features can I use in Wakanda backend ?

  • Wakanda uses V8 4.2 as its JavaScript engine.
  • To check if a feature is supported in Wakanda Server, look it up in the most recent 6.x column of this document.
  • You can use synchronous and asynchrnous features of ES only inside a NodeWorker.
  • You can use synchronous features everywhere in Wakanda Server.
