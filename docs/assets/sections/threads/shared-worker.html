Shared Worker

What is it

  • A shared worker runs some code in a separate thread.
  • You can communicate with a SharedWork from anywhere in your application.

How to

The SharedWorker API needs the following :

  • A JavaScript file that contains the code to be run.
  • An ID to identify the SharedWorker instance.
You can use the same file to create multiple SharedWorkers (multiple threads) but you need to use a different ID for every new SharedWorker you cant to create.
